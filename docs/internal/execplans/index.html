LumaGo ExecPlans

Ordered execution plans derived from the internal Go + Vulkan 2D game engine roadmap. Each plan contains task and acceptance checklists.

  1. 00 - PC Vulkan FoundationOpen a desktop window, initialize Vulkan, and render one textured quad.
  2. 01 - Sprite Engine and BatchingRender sorted, batched sprites with atlas support and camera transforms.
  3. 02 - Normal-Mapped 2D LightingAdd color, normal, light accumulation, and composite passes.
  4. 03 - Per-Light 2D Shadow MapsGenerate hard shadows from occluders for shadow-casting lights.
  5. 04 - SDF-Based Experimental ShadowsPrototype toggled SDF raymarched soft shadows.
  6. 05 - Polish and PerformanceAdd overlays, metrics, config, documentation, and MVP acceptance validation.
